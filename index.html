Introduction

JavaScript is a versatile, cross-platform, object-oriented scripting language commonly used for web development. It is lightweight and integrates well within host environments, such as web browsers, to provide programmatic control over web page elements.
JavaScript includes a standard library of objects like Array, Date, and Math, and a fundamental set of language constructs including operators, control structures, and statements. The core JavaScript language can be extended for various applications through additional objects.

  • Client-side JavaScript: Extends the core language to manage browser behavior and the Document Object Model (DOM). This allows manipulation of HTML elements, handling user interactions, and responding to events like clicks and form submissions.
  • Server-side JavaScript: Expands JavaScript capabilities for server environments, enabling database communication, session management, and file operations on the server.

What You Should Already Know

This guide assumes you have:

  • A basic understanding of the Internet and the World Wide Web (WWW).
  • Familiarity with HyperText Markup Language (HTML).
  • Some programming experience. If you are new to programming, consider exploring introductory JavaScript tutorials.

JavaScript vs Java

While JavaScript and Java share some similarities, they have fundamental differences. JavaScript resembles Java in syntax but differs in several key areas:

  • Typing: JavaScript is dynamically typed and does not enforce static type checking as Java does.
  • Object Model: JavaScript uses a prototype-based model, providing dynamic inheritance, whereas Java uses a class-based model with compile-time class definitions.
  • Declaration: JavaScript allows for a more flexible, free-form approach without strict declarations for variables, functions, or methods.

Hello World

To begin with JavaScript, try writing a simple "Hello World" program: 

function greetMe(yourName) {
  alert("Hello " + yourName);
}
greetMe("World");

Variables

Variables in JavaScript are symbolic names for values. Identifiers must start with a letter, underscore (_), or dollar sign ($), and can include digits after the first character. JavaScript is case-sensitive, allowing identifiers like Number_hits, temp99, and _name

Declaring Variables

You can declare variables in three ways:

  • With var: var x = 42;
    This syntax declares both local and global variables. 
  • By Assignment: x = 42;
    This creates a global variable and is generally discouraged. 
  • With let: let y = 13;
    This declares a block-scoped local variable.

Variable Scope

Variables declared outside any function are global, while those declared within a function are local. JavaScript before ECMAScript 2015 had function scope, but with ECMAScript 2015, let introduced block scope: 

if (true) {
  var x = 5; // function or global scope
}
console.log(x); // 5

if (true) {
  let y = 5; // block scope
}
console.log(y); // ReferenceError: y is not defined

Global Variables

Global variables are properties of the global object (window in browsers). You can access them using window.variable and from different windows or frames using syntax like parent.phoneNumber.

Constants

Use const to declare a read-only variable:

const PI = 3.14;

Constants must be initialized at declaration and cannot be reassigned. They follow the same scope rules as let variables.

Data Types

JavaScript defines seven data types:

  • Primitive Types:
    • Boolean: true, false
    • Null: null
    • Undefined: undefined
    • Number: 42, 3.14159
    • String: "Hello"
    • Symbol (new in ECMAScript 2015): Unique and immutable
  • Object: Used to store collections of data and more complex entities.

Control Flow

  • if...else Statement: Executes code based on a condition: 
    
if (condition) {
  statement_1;
} else {
  statement_2;
}
    You can chain multiple conditions with else if: 
    
if (condition_1) {
  statement_1;
} else if (condition_2) {
  statement_2;
} else {
  statement_last;
}
  • while Statement: Repeats execution while a condition is true: 
    
while (condition) {
  statement;
}
    Example: 
    
var n = 0;
var x = 0;
while (n < 3) {
  n++;
  x += n;
}

Function Declarations

Define a function with the function keyword:


function square(number) {
  return number * number;
}

Functions can take arguments and return values. Primitive values are passed by value, while objects are passed by reference.
